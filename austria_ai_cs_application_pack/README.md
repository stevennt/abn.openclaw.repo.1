# Austria University Application Pack (AI & Computer Science)

This folder is a complete starter pack for an 11th-grade student preparing to apply to universities in Austria for AI/Computer Science related programs.

## Folder Structure

- `01_Profile/` → student profile + achievements
- `02_University_List/` → shortlist of Austrian universities and target programs
- `03_Application_Documents/` → motivation letter, CV, personal statement, study plan
- `04_Recommendations/` → recommendation request templates + referee notes
- `05_Timeline/` → month-by-month application timeline
- `06_Submission/` → final submission checklist + email template

## How to Use

1. Fill in placeholders marked like `[YOUR_NAME]`.
2. Start with `01_Profile/student_master_profile.md`.
3. Customize one motivation letter per university.
4. Export final docs to PDF before submitting.
5. Track every requirement in `06_Submission/final_checklist.md`.

## Notes

- This pack is guidance/template material, not legal advice.
- Always verify each university’s official admissions page for the current intake.
