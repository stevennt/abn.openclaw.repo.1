# Final Submission Checklist

## Core Documents
- [ ] Passport copy
- [ ] Academic transcripts (official)
- [ ] Diploma/provisional graduation letter (when available)
- [ ] CV (PDF)
- [ ] Motivation letter (university-specific)
- [ ] Personal statement
- [ ] Study plan
- [ ] Recommendation letters
- [ ] Language certificate(s)

## Quality Check
- [ ] Name and date of birth are consistent across all docs
- [ ] File names are professional (e.g., `Nguyen_An_CV.pdf`)
- [ ] No grammar/spelling issues
- [ ] University name/program name correctly customized

## Submission Check
- [ ] Uploaded all required files
- [ ] Confirmed application fee payment (if required)
- [ ] Saved submission confirmation email/screenshots
- [ ] Added reminder for follow-up deadlines

## Suggested File Naming
- `[Name]_CV.pdf`
- `[Name]_Motivation_[University].pdf`
- `[Name]_PersonalStatement.pdf`
- `[Name]_StudyPlan.pdf`
- `[Name]_Transcript.pdf`
- `[Name]_Recommendation_[Teacher].pdf`
